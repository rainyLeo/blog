FormData, Blob, ArrayBuffer

FormData

FormData 接口可以很方便地创建表示表单字段和值的键值对, 并使用XMLHttpRequest的.send() 来发送. 它的格式和 编码类型 为 multipart/form-data 的一样

  1. 构造函数
    new FormData (form? : HTMLFormElement)

  2. 方法
    append()
    delete()
    entries()
    has()
    set
    values()

Blob

Blob 对象表示不可变的类似文件对象的原始数据。Blob 中的数据不一定是 JavaScript原生形式的。File 接口基于Blob，继承了 blob的功能并将其扩展使其支持用户系统上的文件。(Binary Large Object)

使用 Blob() 构造函数可以从其他非 blob对象和数据构造一个Blob。要创建包含另一个 blob数据的子集 blob，可以使用 slice()方法。要获取用户文件系统上文件的 Blob对象，参考 File文档。

  1. 构造函数

Blob(blobParts[, options])
返回一个新创建的 Blob 对象，其内容由参数中给定的数组组成。

  1. 属性

Blob.size 只读
Blob 对象中所包含数据的大小（字节）。
Blob.type 只读
一个字符串，表明该 Blob对象所包含数据的 MIME类型。如果类型未知，则该值为空字符串。

  1. 方法

Blob.slice([start,[ end ,[contentType]]])
返回一个新的 Blob对象，包含了源 Blob对象中指定范围内的数据。
可用于文件分片上传

  • 使用 Blob 创建一个Object URL

  • 从 Blob 中提取数据

从Blob中读取内容的唯一方法是使用 FileReader
