水平居中

  1. inline-* 元素

    • 父 block 元素设置 text-align: center

  2. 1个block 元素

    • 设置 width(不设置的话等于父元素的 width), 水平 margin 设为 auto

  3. 多个 block元素

    • 设置 display: inline-block;
    • 用 flex, 父元素 { display: flex; justify-content: center;}

  1. inline-* 元素:

    • 单行
    1. 上下 padding 相同
    2. line-height 等于 height

    1. 上下 padding 相同
    2. { display: table-cell; vertical-align: middle;}, 父 block 元素 display: table
    3. 用 flex, 父 block 元素 {display: flex; flex-direction: column; justify-content: center; resize: vertical; overflow: auto}
    4. 用伪元素, div{position: relative;} div:before{ content: ' '; display: inline-block; height: 100%: width: 1%; vertical-align: middle;} p {display: inline-block; vertical-align: middle}
